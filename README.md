# Contributing
---------------

 - Install `docker`.
 - Install `docker-compose`.
 - Install `poetry`.
 - Install `tox`.
 - Run `pre-commit install`.
 - Create and branch and make changes.
 - execute `tox -e py38`
 - Hooks will automatically run; NO PR's will be accepted with a decrease in coverage.
